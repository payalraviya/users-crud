### User Crud

setup

- clone repo
```
git@github.com:payalraviya/users-crud.git
```

- install dependencies
```
npm install
```

- copy .env.example to .env
```
cp .env.example .env
```

- run locally
```
npm run dev
```

- run with docker
```
docker compose up
```
